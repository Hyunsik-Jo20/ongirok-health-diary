ONGIROK GUIDE

온기록 사용 매뉴얼

1. AI API 키 발급

  1. OpenAI는 platform.openai.com에서 로그인한 뒤 API Keys 메뉴에서 새 키를 발급합니다.
  2. Gemini는 Google AI Studio에서 Get API key를 눌러 키를 만듭니다.
  3. 앱의 설정 → 분석 AI에서 OpenAI 또는 Gemini를 고르고 키를 붙여넣습니다.
  4. 키는 이 스마트폰 브라우저에만 저장되며, 다른 사람과 공유하지 마세요.

2. 공공데이터포털 날씨 API 신청

  1. data.go.kr에 로그인하고 검색창에 “단기예보” 또는 “기상청 단기예보”를 검색합니다.
  2. 서비스 목록에서 “기상청_단기예보 ((구)_동네예보) 조회서비스” 계열 서비스를 선택합니다.
  3. 활용신청을 누르고 사용 목적을 “개인 건강 일기 날씨 참고”처럼 적어 신청합니다.
  4. 승인 후 마이페이지 → 활용신청 현황에서 일반 인증키를 복사합니다.
  5. 앱 설정의 공공데이터포털 API 키에 붙여넣고, 날씨 지역을 선택하거나 현재 위치로 설정합니다.

공공데이터 키도 개인 키입니다. 앱에는 입력해도 되지만, 게시판·깃허브·캡처 이미지에 노출하지 않는 편이 안전합니다.

3. 건강 일기장 사용 방법

  1. 표지에서 기본 건강정보를 입력하거나 병원 기록·검진 기록 파일을 가져옵니다.
  2. 1쪽에는 제목, 자유 일기, 걸음·수면·수분, 운동/식사 사진 또는 파일을 기록합니다.
  3. 2쪽에서 분석하기를 누르면 AI가 글·수치·첨부 이미지·날씨를 함께 읽고 건강 리포트를 만듭니다.
  4. “보완하면 더 정확해져요”에 나온 항목은 버튼을 눌러 해당 입력 위치로 바로 추가할 수 있습니다.
  5. 하단 캘린더·인사이트·나의 기록에서 날짜별 기록과 추세를 확인합니다.

4. 데이터 백업·복원

스마트폰을 바꾸거나 다른 브라우저에서 쓰려면 먼저 백업 파일을 저장하고, 새 기기에서 다시 업로드하세요.

  • 백업 포함: 기본 건강정보, 일기 텍스트, 수치 기록, 분석 결과, 프롬프트/지역 설정
  • 백업 제외: OpenAI·Gemini·공공데이터 API 키, 원본 사진 파일
  • 복원 후에는 설정창에서 API 키만 다시 입력하면 됩니다.
